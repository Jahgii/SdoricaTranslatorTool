# Sdorica Translator Tool (STT)
Tool to modify the majority of in-game text, excluding text located outside the game files.

## Development
To Run Frontend with Wails
```sh
cd Wails && wails dev
```

To Run Frontend
```sh
cd Angular && ng serve
```

To Run Dotnet
```sh
cd Dotnet && dotnet run
```

## Build
Wails
```sh
cd Wails && make windows
```
